Python 스크립트 수정 및 실행 파일 업데이트 가이드
1. Python 스크립트 수정
원하는 텍스트 편집기나 IDE를 사용하여 ptqtest.py 파일을 열고 필요한 변경사항을 적용하세요.
2. 기존 빌드 파일 제거
이전 빌드로 생성된 임시 파일 및 디렉터리를 제거하여 새로운 빌드 시 문제가 발생하지 않도록 합니다.

```bash
rm -rf /Users/jeondonghwan/Desktop/thefish_src/test/build
rm -rf /Users/jeondonghwan/Desktop/thefish_src/test/dist
rm -rf /Users/jeondonghwan/Desktop/thefish_src/test/ptqtest*.spec
```

3. PyInstaller로 다시 패키징
수정된 Python 스크립트를 pyinstaller로 다시 패키징하여 실행 파일을 생성합니다.

```bash
cd /Users/jeondonghwan/Desktop/thefish_src/test
pyinstaller yourscript.py ptqtest1.py
```
4. 새로운 실행 파일 확인
새로 생성된 실행 파일을 실행하여 업데이트가 제대로 적용되었는지 확인합니다.

```bash
./dist/ptqtest/ptqtest
```
